<area /> semantisch Verweis-sensitiver Bereich in einer Image-Map-Grafik block
<b></b> nicht semantisch Stilistische Hervorhebung im Text (üblicherweise fett dargestellt) inline
<base /> semantisch Basis-URL für Referenzen inline
<bdo></bdo> semantisch Richtungsänderung bei bidirektionalem Text inline
<big></big> nicht semantisch Großschrift inline
<blockquote></blockquote> semantisch Zitat in Form eines eigenen Textabsatzes block
<body></body> semantisch Inhaltsbereich eines HTML-Dokuments block
<br /> semantisch Erzwungener Zeilenumbruch an der betreffenden Stelle inline
<button></button> semantisch Schaltfläche in einem Formular inline
<caption></caption> semantisch Tabellenüberschrift / -unterschrift inline
<cite></cite> semantisch Zitat innerhalb eines Absatzes inline
<code></code> semantisch Quelltext-Auszug innerhalb eines Absatzes inline
<col /> semantisch Tabellenspalte block
<colgroup></colgroup> semantisch Gruppe von Tabellenspalten block
<dd></dd> semantisch Definition, Beschreibung, detaillierter Inhalt zu einem dt-Element inline
<del></del> semantisch gelöschter Inhalt (Änderungsmarkierung) block
<dfn></dfn> semantisch Definition innerhalb eines Absatzes inline
<div></div> semantisch Allgemeiner Bereich block
<dl></dl> semantisch Liste mit dt- und dd-Elementen, für Definitionslisten, Dialoge usw. block
<dt></dt> semantisch Zu definierender Ausdruck in einer Definitionsliste, Sprechername in einem Dialog usw. inline
<em></em> semantisch emphatischer Ausdruck innerhalb eines Absatzes (z.B. überspitzte Formulierung) inline
<fieldset></fieldset> semantisch Gruppe von Elementen in einem Formular, Reiter in einem Dialog block
<form></form> semantisch Formular, Dialog block
<h1-h6></h1-h6> semantisch Überschrift 1. bis 6. Ordnung block
<head></head> semantisch Kopfbereich eines HTML-Dokuments block
<hr></hr> semantisch Trennlinie block
<html></html> semantisch Dokumentelement von HTML-Dokumenten, schließt das gesamte Dokument ein block
<i></i> semantisch spezieller Ausdruck im Text, z.B. Fachterminus oder Idiom (üblicherweise kursiv dargestellt) inline
<img /> semantisch Referenzierte und eingebettete Grafik inline
<input /> semantisch Formularelement, verwendbar für diverse Typen wie Eingabefelder, Schaltflächen, Auswahlfelder usw. inline
<ins></ins> semantisch Eingefügter Inhalt (Änderungsmarkierung) inline
<kbd></kbd> semantisch Darstellung eines über Tastatur einzugebenden Textes im Text inline
<label></label> semantisch Beschriftungstext eines Formularfeldes inline
<legend></legend> semantisch Überschrift für eine fieldset-Gruppe von Elementen in einem Formular inline
<li></li> semantisch Listenpunkt in einer ul-, ol- oder menu-Liste block
<link /> semantisch Logische Beziehungen der aktuellen Webseite zu anderen Webseiten inline
<map></map> semantisch Bereich für verweis-sensitive Grafiken inline
<meta></meta> semantisch Meta-Angabe für nicht im Inhalt sichtbare Dokumentinformationen oder für Anweisungen an den Webserver inline
<noscript></noscript> semantisch Bereich für Inhalte, die angezeigt werden, wenn JavaScript im Browser deaktiviert oder nicht verfügbar ist block
<object></object> semantisch Eingebettetes Objekt, geeignet für spezielle Multimedia-Objekte, Java-Applets usw. inline
<ol></ol> semantisch Geordnete (z.B. nummerierte) Liste block
<optgroup></optgroup> semantisch Gruppe von Auswahlmöglichkeiten in einer Auswahlliste block
<option></option> semantisch Auswahlmöglichkeit in einer Auswahlliste block
<p></p> semantisch Paragraph, Absatz block
<param /> semantisch Parameter eines Objekts bei object-Element inline
<pre></pre> nicht semantisch Präformatierter Text (alle Whitespace-Zeichen werden im Browser so dargestellt wie im Text enthalten) block
<q></q> semantisch Wörtliche Rede in einem Text inline
<samp></samp> semantisch Beispieltext in einem dokumentierenden Text inline
<script></script> semantisch Scriptbereich für JavaScript und andere Scriptsprachen inline
<select></select> semantisch Auswahlliste in einem Formular inline
<small></small> nicht semantisch kleiner dargestellter Text als Normalschrift inline
<span></span> semantisch allgemeiner Bereich innerhalb eines Absatzes inline
<strong></strong> semantisch stark betonter Inhalt im Text inline
<style></style> semantisch Bereich für Stylesheet-Definitionen, z.B. in CSS inline
<sub></sub> semantisch Tiefgestellter Text inline
<sup></sup> semantisch Hochgestellter Text inline
<table></table> semantisch Tabelle block
<tbody></tbody> semantisch Tabellenkörper mit den eigentlichen Daten block
<td></td> semantisch Tabellenzelle block
<textarea></textarea> semantisch Mehrzeiliger Eingabebereich in einem Formular inline
<tfoot></tfoot> semantisch Fußbereich in einer Tabelle, z.B. für summarische Daten block
<th></th> semantisch Tabellenkopfzelle block
<thead></thead> semantisch Kopfbereich einer Tabelle block
<title></title> semantisch Titel eines HTML-Dokuments block
<tr></tr> semantisch Tabellenzeile block
<tt></tt> semantisch »Fernschreiber«-Text inline
<ul></ul> semantisch Unsortierte Liste (Aufzählungsliste) block
<var></var> semantisch Variabler Name in einem dokumentierenden Text inline
